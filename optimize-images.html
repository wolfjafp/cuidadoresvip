Optimizador de Imágenes para CuidadoresVIP

Instrucciones:

Esta herramienta te ayudará a optimizar las imágenes de tu sitio web para mejorar el rendimiento:

  1. Haz clic en "Analizar Imágenes" para ver todas las imágenes del sitio
  2. Selecciona las imágenes que deseas optimizar
  3. Haz clic en "Optimizar Seleccionadas" para convertirlas a WebP
  4. Descarga las imágenes optimizadas y reemplaza las originales

Nota: Esta herramienta utiliza la API de navegador para optimizar las imágenes, por lo que no necesita instalación adicional.
